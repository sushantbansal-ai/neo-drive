# Nevo Test Drive

## Overview

This project implements a test-drive booking service for EV vehicles. It includes a NestJS backend, a React/Vite frontend, PostgreSQL persistence through Prisma, seed data, and a root `dev.sh` script intended to stand the project up for review in one step.

## Running Locally

From the repository root:

```bash
./dev.sh
```

Default local URLs:

- Web app: `http://localhost:5173`
- Backend API: `http://localhost:3000`
- Swagger docs: `http://localhost:3000/docs`
